使い方

  1. Issueのタイトルにチケットタイトル - 2.5Hのフォーマットで作業工数（時間）を記載してください。(例)
  2. Issueに担当者をアサインしてください。
  3. フォームにリポジトリのオーナー名とリポジトリ名を入力してIssueを取得します。たとえばhttps://github.com/deer-inc/github-manager-example-repositoryの場合ownerはdeer-incでリポジトリ名はgithub-manager-example-repositoryになります。
  4. コラボレータごとの工数が算出されます。
  5. 算出された工数をもとに請求書を作成してください。（請求書作成機能はありません）
